Unicode 編碼/解碼

使用說明

  • 編碼: 將文字轉換為 Unicode 編碼格式（\\uXXXX）
  • 解碼: 將 Unicode 編碼轉換回文字
  • 支援所有 Unicode 字符

Base64 編碼/解碼

使用說明

  • 編碼: 將文字轉換為 Base64 編碼格式
  • 解碼: 將 Base64 編碼轉換回文字
  • 自動雙向轉換，支援中文和特殊字符

URL 編碼/解碼

使用說明

  • 編碼: 將文字轉換為 URL 編碼格式（適用於 URL 參數）
  • 解碼: 將 URL 編碼轉換回文字
  • 自動雙向轉換，支援中文和特殊字符
