# 開發工具箱 (Dev Toolbox)

一個簡單的前端開發者工具箱，提供各種實用的編碼/解碼工具。

## 功能

### 1. Unicode 編碼/解碼
- **編碼**: 將文字轉換為 Unicode 編碼格式（`\uXXXX`）
- **解碼**: 將 Unicode 編碼轉換回文字
- **複製**: 一鍵複製結果到剪貼板
- **自動轉換**: 雙向自動轉換

### 2. Base64 編碼/解碼
- **編碼**: 將文字轉換為 Base64 編碼格式
- **解碼**: 將 Base64 編碼轉換回文字
- **複製**: 一鍵複製結果到剪貼板
- **自動轉換**: 雙向自動轉換，支援中文和特殊字符

### 3. URL 編碼/解碼
- **編碼**: 將文字轉換為 URL 編碼格式（用於 URL 參數）
- **解碼**: 將 URL 編碼轉換回文字
- **複製**: 一鍵複製結果到剪貼板
- **自動轉換**: 雙向自動轉換，支援中文和特殊字符

### 4. JSON 格式化
- **格式化**: 將 JSON 格式化為易讀的縮進形式（2 空格縮進）
- **壓縮**: 將 JSON 壓縮為單行形式
- **複製**: 一鍵複製結果到剪貼板
- **驗證**: 自動驗證 JSON 格式，錯誤會顯示在輸出框

## 使用方法

1. 打開 `index.html` 文件在瀏覽器中
2. 輸入要編碼或解碼的內容
3. 點擊相應的按鈕（編碼/解碼）
4. 結果會顯示在下方的輸出框
5. 點擊「複製結果」按鈕將結果複製到剪貼板

### 快捷鍵
- **Ctrl + Enter**: 編碼
- **Ctrl + Shift + Enter**: 解碼
